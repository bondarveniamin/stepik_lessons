Предусловия:
Адрес интернет-магазина - http://selenium1py.pythonanywhere.com/ru/
открыть браузер
Шаги:
- Шаг 1, перейти по ссылке на основную (главную) страницу, ожидаемый результат: открыта главная страница (отображается название магазина - "Oscar Sandbox");

Добавление товара в корзину 2.1.1
Предусловия:
Адрес каталога - http://selenium1py.pythonanywhere.com/ru/catalogue/
открыть браузер
Шаги:
- Шаг 1, перейти по ссылке на страницу с заголовком "Все товары", ожидаемый результат: отображаются карточки с товарами;
- Шаг 2, найти карточку товара с названием "The shellcoder's handbook", ожидаемый результат: карточка с таким названием отображается на странице;
- Шаг 3, нажать кнопку "Добавить в корзину", ожидаемый результат: обновилась страница, появилась текстовая строка - "The shellcoder's handbook был добавлен в вашу корзину.".

Поиск товара с помощью поля "Найти" 2.2.1
Предусловия:
Адрес каталога - http://selenium1py.pythonanywhere.com/ru/catalogue/
открыть браузер
Шаги:
- Шаг 1, перейти по ссылке на страницу с заголовком "Все товары", ожидаемый результат: отображаются карточки с товарами;
- Шаг 2, найти на странице текстовое поле "Найти", ожидаемый результат: отображается текстовое поле "Найти";
- Шаг 3, ввести в поле "Найти" строку "The shellcoder's handbook", ожидаемый результат: в текстовом поле отображается строка "The shellcoder's handbook";
- Шаг 4, нажать кнопку "Найти", ожидаемый результат: на странице отображается карточка с товаром "The shellcoder's handbook".
